# Drawmad_2.0

Drawmad is a design application.

### Details

The goal of the project is to create a simple application for designing.

### Prerequisites
- `node v18.12.1`
- `npm 8.19.2`

### Setup

Install dependencies:

```shell
npm install
```

Run app:

```shell
npm run dev
```

## Project details

TypeScript project setup using Vite.

Testing using Vitest, JSDOM and RTL.
